Space Blox

The goal of the game is to break all the (breakable) blocks on the screen. Blocks can be broken by colliding with the ball. Some more things to know before you play:

  • You have three lives.
  • Hitting each breakable block earns you 10 points.
  • There are four levels to beat!
  • You can use the mouse OR the left/right arrow keys to control the paddle.
  • Each level has two powerups. Breaking a green block will unlock that powerup (and give you 25 points!) - it may help you or it may increase the challenge!
  • Grey, unbreakable, blocks will start appearing in Level 2. Hitting them earns NO points.
